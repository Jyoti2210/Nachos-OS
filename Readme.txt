This repository contains implementation of the following Operating System concepts using Nachos OS in C++:

#Lab 1:
1) Multithreading and Multiprogramming
2) Preemptive Priority Scheduling 

#Lab 2:
1) Implementation of following System_Calls:
	- FORK
	- WAIT
	- WRITE
	- EXIT
2) Virtual Memory
3) FIFO Page Replacement Algorithm
4) Round Robin Scheduling Algorithm

#Nucleus_Message Passing System:
Implemented multiprogamming and communication between different processes coordinated by the message buffering process within the system nucleus.
